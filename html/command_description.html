功能指令

  • /help: 查看指令分類列表
  • /about: 機器人簡介
  • /anime: 查詢最新動漫資訊指令
  • /botinfo: 查看機器人資訊
  • /serverinfo: 查看伺服器資訊
  • /userinfo <@用戶、用戶ID>: 查看用戶資訊
  • /avatar <@用戶、用戶ID>: 查看用戶頭像
  • /invite: 邀請機器人
  • /say <訊息>: 發送一般式訊息
  • /embed: 發送嵌入式訊息
  • /giveaway: 機器人抽獎系統
  • /instagram <用戶名稱>: 查看Instagram帳戶資訊
  • /youtube_channel_info: 獲取YT頻道資訊
  • /youtube_video_info: 獲取YT影片資訊
  • /emoji <表情符號>: 獲取表情符號
  • /url <網址>: 將網址進行縮短指令
  • /qrcode <網址>: 將網址轉換成QRcode指令
  • /suggest <訊息>: 提供建議給開發者
  • /support: 投票支持白玥機器人

管理指令

  • /help: 查看指令分類列表
  • /warning: 機器人警告系統
  • /clear <1~100>: 清除該頻道的訊息，可指定要清除的訊息數量。
  • /kick <@用戶、用戶ID>: 將成員從伺服器踢出。
  • /ban <@用戶、用戶ID>: 將成員從伺服器中停權。
  • /mute <@用戶、用戶ID>: 禁言成員於指定頻道。
  • /unmute <@用戶、用戶ID>: 解除禁言成員於指定頻道。
  • /poll <頻道> <題目> <選項1> <選項2>...: 機器人投票系統
